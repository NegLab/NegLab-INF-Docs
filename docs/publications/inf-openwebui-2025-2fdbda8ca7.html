OpenWebUI and Local LLM Hosting

Providing access to locally hosted large language models for the CRC

Leon Hammerla, Giuseppe Abrami

2025

Visit project Code & archive

Abstract

We set up and host an OpenWebUI instance for the CRC, providing researchers with access to large language models running on local servers. The infrastructure enables the use of generative AI models within the CRC while keeping model execution and data processing on institutional infrastructure.
